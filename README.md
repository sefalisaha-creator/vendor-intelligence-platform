# vendor-intelligence-platform
PlatformA centralized, single source of truth for organization-wide vendor discovery, collaboration, governance, and historical footprints. This platform is built specifically to handle the relationship management layer for an enterprise operating across multiple cities, driving cross-department collaboration and preventing duplicate vendor onboarding.  

Note: This platform focuses entirely on visibility, risk evaluation, and knowledge sharing—it is intentionally not built for procurement transactions or accounting processes.  

🚀 Tech StackFrontend: 
React.js (Vite), Tailwind CSS, Lucide React, React Router, 
AxiosBackend: Node.js, Express.js (CommonJS / require syntax architecture)
Database: MongoDB via Mongoose Object Modeling
